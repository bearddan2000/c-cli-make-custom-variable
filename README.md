# c-cli-make-custom-variable

## Description
This is a POC project to demonstrate a
make calling `echo` command with custom variable.

## Tech stack
- make
- echo

## Docker stack
- alpine:edge

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
